# ZETA — Engagement Layer for Zolve

**INFORMS × Zolve Hackathon · April 2026**

## Problem
Financial apps are transactional. Users open them to pay, then leave. For Zolve's international student users, this means missing the chance to educate, build habits, and retain.

## Solution
ZETA is an engagement layer that turns the immigrant financial journey into a living product. Three reinforcing loops:
- **Lifecycle awareness** — the app knows your stage (arrival, finding footing, building credit) and adapts
- **Cohort comparisons** — anonymous percentile insights against peers on the same journey
- **AI Coach (Zolvi)** — a culturally aware financial coach powered by Claude, grounded in your real transactions

## Features
- **ZETA Home** — lifecycle stage banner, AI nudge card, quick actions, cohort snapshot, daily insight
- **Zolvi** — real-time Claude Sonnet streaming, culturally aware, knows your actual data
- **Credit Quests** — gamified credit-building with XP, levels, badges
- **Cohort** — percentile rankings + group savings challenges
